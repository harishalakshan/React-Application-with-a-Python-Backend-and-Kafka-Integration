# Full Stack React + Flask + Kafka Project

This project demonstrates a full-stack application using:

- **React** for the frontend UI
- **Flask** for the Python backend
- **Apache Kafka** for messaging

## Usage

1. Start Kafka and Zookeeper
2. Run Flask backend:
    ```bash
    cd backend
    python app.py
    ```
3. Start frontend:
    ```bash
    cd frontend
    npm install
    npm start
    ```
